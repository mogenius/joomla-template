# Joomla template
This is a Dockerfile template to run Joomla on [mogenius](https://mogenius.com). We use it to deploy Joomla from our service library. If you have a Joomla project that you want to run on mogenius feel free to add this Dockerfile to your project.
## Deployment on mogenius
[mogenius](https://mogenius.com) is the easiest way to run Joomla in the cloud. It's free, so [sign up now](https://studio.mogenius.com/user/registration) and deploy your app with mogenius.
